To execute our website on your own computer, first you will need git, and pyhton which you can install from their websites . Then type following commands on cmd

mkdir Vespera_Website
cd Vespera_Website
git clone https://github.com/Ayz345/Vespera.git
cd Vespera
python -m venv venv
venv\scripts\activate
pip install -r requirements.txt
python App.py
